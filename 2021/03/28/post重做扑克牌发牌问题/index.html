扑克牌问题

要求：52张扑克牌中给4位玩家各发13张牌，然后将4位玩家手中的13张牌分别从小到大理牌后输出。说明：假定梅花最小，方块次小，红桃次大，黑桃最大。

typedef struct
{ int suit; // 花色 1..4
